## 1. 开场（1 分钟）

**口述要点：**

1. 产品名：**FusionBid 智标聚合助手**  
2. 用户用**自然语言**描述需求 → 系统解析意图 → 多源采集 → 清洗去重增量 → **Word 报告**  
3. 数据源：**≥2 公开源**（ccgp、cebpub）+ **1 登录态源**（Playwright 手动登录）  
4. **无硬编码招标结果**；演示数据会标注 `data_mode`  

打开首页，展示健康检查 `ok`、阶段信息。

---

## 2. 示例一：立即执行（3 分钟）

**输入（新建检索）：**

```text
最近1个月的安徽省区域内的服务器招标信息都有哪些
```

**演示步骤：**

1. 点击「解析意图」  
2. 指出识别结果：关键词「服务器」、区域「安徽省」、最近 1 个月起止日期、**立即执行**、无定时  
3. 必要时手工微调后「确认并创建任务」  
4. 任务列表 → **立即执行**（提醒：ccgp 限速，可能需数十秒）  
5. 打开「采集结果」：展示 `live` 标注、来源链接  
6. 打开「报告中心」→ 下载 Word，打开封面/条件/统计/结果  

**话术：** 报告文件名保留用户原问题 + 时间戳，不覆盖历史文件。

---

## 3. 示例二：指定月份（2 分钟）

**输入：**

```text
2026年3月份的上海区域内的充电桩招标信息都有哪些
```

**要点：**

- 日期范围 **2026-03-01～2026-03-31**  
- 区域 **上海市**，主题 **充电桩**  
- 立即执行  

可只做到解析确认页（若时间紧），说明与示例一执行路径相同。

---

## 4. 示例三：每日定时 + 增量（3 分钟）

**输入：**

```text
最近3个月的上海区域内的充电桩招标信息都有哪些，请汇总后每天9:00发送给我
```

**要点：**

- 定时：`daily` `09:00`，`execute_immediately=false`  
- 确认后任务状态 `scheduled`  
- 展示「暂停 / 恢复」  
- 点击「立即执行」模拟一次定时产出  
- **再次立即执行**：说明增量——已推送不再进报告，仅新增/内容变化  

**话术：** 失败执行不会把未交付内容标成已推送。

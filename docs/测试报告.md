## 5. 已知限制

| 项 | 说明 | 缓解 |
|----|------|------|
| 站点改版 | 选择器/接口可能失效 | 健康检查、适配器隔离 |
| 限流 | ccgp 严格限速 | 默认间隔 ≥4s、限制页数 |
| 登录门户结构差异 | 默认 URL 可配置 | `.env` 自定义门户 |
| 真实站联调环境差异 | 演示网络影响结果数量 | 报告仍生成；可说明增量为空 |

## 6. 结论

系统在**自动化测试通过**的前提下，具备赛题要求的主链路可运行能力。详情未验证时只保留元数据，不将其误写为公告正文；“未去重完整快照”保留本轮每个来源的合格记录并标注可能重复项，后台入库与 DeliveryHistory 仍保持去重。
真实站点联调结果随网络与站点状态变化，**不得**用静态伪造列表充当 live 验收。

**测试负责人建议结论：通过（功能测试） / 真实采集视环境为有条件通过。**
